Algoritmo de cálculo de média ponderada: um algoritmo que calcula a 
média ponderada de uma lista de números com base nos pesos fornecidos.

1. leia lista_numeros
2. leis lista_pesos
3. var somatorio = 0
4. var somatorio_pesos = 0 
5. para cada elemento da lista_numeros, faça:
6. somatorio = somatorio + (elemento_atual * peso correspondente)
7. para cada elemento da lista_pesos, faça:
8. 	somatorio_pesos = somatorio_pesos + elemento_atual
9. resultado = somatorio/somatorio_pesos
10. escreva resultados
